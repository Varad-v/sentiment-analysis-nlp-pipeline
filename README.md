# Sentiment Analysis using NLP Pipeline and Machine Learning Models

## 📌 Project Overview
This project demonstrates an end-to-end implementation of Sentiment Analysis using Natural Language Processing (NLP) techniques and Machine Learning models.

The pipeline processes raw text data, applies preprocessing techniques, converts text into numerical features, and trains multiple models to classify sentiment.

---

## ⚙️ NLP Pipeline
Raw Text → Preprocessing → Feature Engineering → Model Training → Evaluation

---

## 🔧 Preprocessing Steps
- Lowercasing
- Removal of punctuation
- Stopword removal
- Tokenization
- Lemmatization
- URL removal

---

## 📊 Feature Engineering
- Bag of Words (BoW)
- TF-IDF

---

## 🤖 Models Used
- Logistic Regression
- Naive Bayes
- Decision Tree

---

## 📈 Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

---

## 🏆 Results
- Logistic Regression performed best
- TF-IDF improved performance significantly
- Decision Tree showed lower performance due to overfitting

---

## 📌 Conclusion
A well-structured NLP pipeline significantly improves sentiment analysis performance. TF-IDF with Logistic Regression provided the best results.
